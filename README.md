# Culture Impact Dashboard

Track community activities, measure business outcomes, and quantify cultural ROI.

## Features

- **Activity Tracking** - Log ERG events, mentorship programs, wellness initiatives, and more
- **Business Outcomes** - Tag activities with outcomes like retention, performance, engagement, hiring, and wellbeing
- **ROI Calculator** - Calculate retained dollar value using industry-standard 1.5x salary replacement cost
- **Executive Narratives** - Generate AI-powered summaries to share your culture impact with leadership

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React + Vite
- CSS custom properties for theming
